    Describe the difference between the print subroutine or method and the println one?
    Your Answer:

    System.out.println() is a method that displays the text passed into its parameter on a new line.

    System.out.print() does the same as System.out.println(), however, it does not make a new line.

    print does not add a line feed (i.e. carriage return) after printing its argument.

    println does add a line feed (i.e. carriage return) after printing its argument.

    Explain why the value of the expression 2 + 3 + "test" is the string "5test" while the value of the expression "test" + 2 + 3 is the string "test23".  What is the value of "test" + 2 * 3?
    Your Answer:

    Operator precedence says that we go from left to right when possible.

    For 2 + 3 + "test" , 2 + 3 is evaulated to 5, then added to the String test. We know it is a String because it is in quotes. Thus the output is 5test.

    2 + 3 + "test" -> 5 + "test" -> "5test"

    "test" + 2 + 3 says to start with a string, changing out output format to a String, then start adding. When we add to a string, it simply adds the value to the end of the String. 

    "test" + 2 + 3 -> "test2" + 3 -> "test23"

    "test" + 2 * 3, Since multiplication takes higher precedence, it is performed before addition. The output is then added to the string/

    "test" + 2 * 3 -> "test" + 6 -> "test6"

     

    Each expression has three operands using the '+' operator.  The precedence of the two instances of '+' is the same, so they are carried out left-to-right (the 'associativity').

    In the first expression the first sub-expression to evaluate, therefore, is '2 + 3'.  The operator '+' for numerical operands is interpreted as addition, so we obtain '5'.  Then we evaluate the second '+' operator, but now with a numeric value (5) and a String value ("test").  Now the '+' operator is interpreted as string concatenation yielding the final result "5test".

    In the second expression, each time the '+' operator is encountered the context is string concatenation, so we obtain "test23"

    In the third expression, the highest priority operator is '*', so first the sub-expression '2 * 3' is evaluated yielding '6'.  Then the string concatenation is performed and the expression evaluates to "test6".
